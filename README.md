# CellarSense.ai

Wedding wine consultation service - helping couples choose the perfect wines for their special day.

## Features

- **Restaurant Mode**: Free wine recommendations for dining out
- **Wedding Mode**: Premium consultation service ($199) with personalized PDF guide

## Tech Stack

- Single-page HTML/CSS/JS application
- No backend required for the main app
- PDF generation via separate Python script (for admin use)

## Deployment

Deployed on Vercel at [cellarsense.ai](https://cellarsense.ai)
